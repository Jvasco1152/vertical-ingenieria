# Screenshots

Esta carpeta contiene las capturas de pantalla del sistema para el README principal.

## Capturas Necesarias

### 1. `dashboard.png`
- **Página**: http://localhost:3000/dashboard
- **Usuario**: admin@vertical.com
- **Contenido**: Dashboard completo mostrando estadísticas y proyectos recientes
- **Resolución**: 1920x1080 o similar
- **Formato**: PNG

### 2. `projects.png`
- **Página**: http://localhost:3000/projects
- **Usuario**: admin@vertical.com
- **Contenido**: Lista de proyectos con filtros visibles
- **Mostrar**: Al menos 3-4 proyectos, panel de filtros expandido
- **Formato**: PNG

### 3. `project-detail.png`
- **Página**: http://localhost:3000/projects/[id]
- **Usuario**: admin@vertical.com
- **Contenido**: Vista detallada de un proyecto con imágenes
- **Mostrar**: Información del proyecto, galería de imágenes, comentarios
- **Formato**: PNG

### 4. `gallery.png`
- **Página**: http://localhost:3000/gallery
- **Usuario**: admin@vertical.com
- **Contenido**: Galería global con grid de imágenes
- **Mostrar**: Al menos 6-8 imágenes en el grid
- **Formato**: PNG

### 5. `users.png`
- **Página**: http://localhost:3000/users
- **Usuario**: admin@vertical.com
- **Contenido**: Tabla de gestión de usuarios

## Instrucciones para Tomar Screenshots

### Opción 1: Herramienta de Recortes de Windows
1. Presiona `Win + Shift + S`
2. Selecciona el área a capturar
3. Guarda como PNG en esta carpeta

### Opción 2: Extensión de Navegador
1. Usa extensiones como "Full Page Screen Capture"
2. Captura la página completa
3. Guarda en esta carpeta con el nombre correspondiente

### Opción 3: DevTools del Navegador
1. Presiona `F12` para abrir DevTools
2. Presiona `Ctrl + Shift + P`
3. Escribe "Capture screenshot"
4. Selecciona "Capture full size screenshot"
5. Guarda en esta carpeta

## Consejos para Mejores Capturas

- ✅ Usa datos realistas (proyectos con nombres significativos)
- ✅ Asegúrate de que haya contenido visible (no pantallas vacías)
- ✅ Usa el zoom del navegador al 100%
- ✅ Oculta información sensible si es necesario
- ✅ Toma capturas con buena iluminación de la UI
- ✅ Evita capturas con errores o estados de carga

## Alternativa: Usar URLs de GitHub Issues

Si prefieres no subir las imágenes al repositorio:

1. Ve a https://github.com/Jvasco1152/vertical-ingenieria/issues/new
2. Arrastra las imágenes al cuadro de texto
3. GitHub generará URLs permanentes
4. Copia las URLs y actualiza el README.md con esas URLs
5. Cierra el issue sin crearlo

Ejemplo:
```markdown
![Dashboard](https://user-images.githubusercontent.com/xxxxx/dashboard.png)
```
